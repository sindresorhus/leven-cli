# leven-cli

> Measure the difference between two strings using the [Levenshtein distance](http://en.wikipedia.org/wiki/Levenshtein_distance) algorithm


## Install

```
$ npm install --global leven-cli
```


## Usage

```
$ leven --help

  Example
    $ leven cat cow
    2
```
